# Inheritance

- class (Superclass)
- more specialized (subclass)
    - inherits data of superclass
    - inherits methods of superclass 
##### ex
- Vehicle (superclass)
    - Motorcycle (Sub Class)
    - Car Class (Sub Class)

- Vehicle
    - Data
        - range
        - top speed
        - passengerCapacity
        - cargoCapacity
    - methods
        - get/set 
- Motorcycle 
    - Data
        - noise
        - handlebarHeight
    - method
        - get/set two member variables above 
- Car
    - Data
        - RearCamera (T/F)
    - Method 
        - get/set members as above
